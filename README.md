### 知识如何驱动决策

AI 做每一个决策，都会先查知识库匹配规则：


## 📡 水质监测：从传感器到决策的完整链路

### 监测什么

虾塘大亨持续监测 **6 项核心水质参数**：

| 参数 | 安全范围 | 危险信号 | AI 反应 |
|------|---------|---------|---------|
| 🌡️ **水温** | 22~28°C | <10°C 或 >32°C | 🚨 Opus 深度分析 + 紧急告警 |
| 💨 **溶解氧 (DO)** | ≥5 mg/L | <3.0 浮头 / <1.5 窒息 | 🚨 立即开增氧机 + 停食 |
| ⚗️ **pH** | 7.5~8.5 | <6.5 酸化 / >9.0 碱中毒 | ⚠️ 投放石灰或有机酸 |
| 🧪 **氨氮** | ≤0.2 mg/L | >0.5 慢性中毒 / >1.0 急性 | 🚨 换水 + 沸石粉 + 停食 |
| 👁️ **透明度** | 30~40 cm | <20 cm 水质恶化 | ⚠️ 换水 + 微生物制剂 |
| ⚖️ **均重** | 持续增长 | 停滞或下降 | 📊 调整投喂 + 检查病害 |

### 三层智能路由

不是每次都调用大模型——AI 会根据情况严重程度自动选择处理方式：

| 情况 | 处理方式 | 响应速度 | 成本 |
|------|---------|---------|------|
| 🟢 一切正常（CSI≤20） | 规则引擎直接决策 | **0ms** | **免费** |
| 🟡 需要关注（CSI 21-60） | Haiku 快速判断 | <2 秒 | 极低 |
| 🔴 紧急危险（CSI>60 或关键词触发） | Opus 深度推理 | <15 秒 | 按需 |

**90% 的日常水质检查由规则引擎处理，不花一分钱 API 费用。** 只有真正需要 AI 深度分析的场景才调用大模型。

### 关键词触发（优先级最高）

有些情况不看评分，直接升级到最高级别处理：

| 触发条件 | 处理级别 | 原因 |
|---------|---------|------|
| 发现死虾 | 🔴 Opus | 可能是 WSSV 病毒，分秒必争 |
| DO < 1.5 | 🔴 Opus | 即将窒息，需要综合判断 |
| 水温极端 | 🔴 Opus | 超出生存范围 |
| pH 致死区 | 🔴 Opus | 直接危及生命 |
| 氨氮 > 1.0 | 🔴 Opus | 急性中毒风险 |

### 历史感知

AI 不只看"这一刻"，还会看"趋势"：

- **12 次历史窗口**：记住最近 12 次的水质数据
- **趋势分析**：DO 是在涨还是在跌？氨氮是在累积吗？
- **2σ 异常检测**：某项指标突然偏离历史均值 2 个标准差 → 自动升级风险

### 传感器接入

| 接入方式 | 适用场景 | 成本 |
|---------|---------|------|
| **仿真模式** | 演示 / 开发 / 测试 | 免费 |
| **涂鸦 IoT** | 真实养殖（爱鱼者 YY-W9909 水质仪） | ¥200-500 |
| **宇树机器狗** | 自动巡塘（DIMOS MCP 接口） | 预留 |
| **无人机** | 多光谱航拍水色分析 | 预留 |

`.env` 里改一行 `SENSOR_MODE=tuya`，就从仿真切到真实传感器，代码零改动。

## 🤖 三个 Agent，三种角色

### 🛡️ 哨兵（Sentinel）— 24 小时守塘人

你不在的时候，哨兵在看着。它每 5 分钟读一次传感器，判断有没有问题，有问题立刻飞书通知你。

**它能发现的问题：**
- 溶解氧骤降（增氧机故障？天气突变？）
- 水温异常（暴雨降温？高温预警？）
- pH 突变（暴雨冲刷？蓝藻爆发？）
- 氨氮超标（投喂过量？底质恶化？）
- 发现死虾（病毒感染？中毒？）
- 蜕壳高峰（需要调整投喂和管理）

**它不会做的事（安全设计）：**
- 不会自动执行清塘、排水、停增氧机等危险操作——只会建议，并标注「⚠️ 需人工确认」
- 不会因为飞书推送失败就停止工作——推送失败只记日志，决策照常进行
- 不会因为 AI 接口超时就卡死——10 秒超时自动切换规则引擎

### 📊 策略（Strategist）— 你的养殖参谋

每天早上 6 点（北京时间），策略 Agent 自动生成一份日报，推送到你的飞书：

- 昨天水质变化趋势（哪些指标在好转，哪些在恶化）
- 投喂执行情况汇总
- 病害风险评估
- 捕捞窗口判断（还要多少天？市场价格怎么样？）
- 今天应该重点关注什么

**它和哨兵的区别：** 哨兵是"消防员"，处理紧急情况；策略是"参谋长"，做中长期规划。

### 📈 增长（Growth）— 帮你找客户赚钱

不只是养好虾，还要卖好虾。增长 Agent 帮你：

**找客户（线索发现）：**
- 从 10 个渠道自动搜索潜在客户（水产协会、天眼查、美团、1688……）
- 自动提取联系信息（手机号、地域、规模）
- ICP 评分（100 分制）筛选高价值线索

**跟客户（触达序列）：**
- Day 1：微信首触（自我介绍 + 价值主张）
- Day 3：发送 ROI 计算报告
- Day 7：发送成功案例
- Day 14：邀请免费试用
- Day 21：试用跟进转付费

**算账（ROI 周报）：**
- 多塘产值对比
- 买家智能匹配（按规格、地域、评分排序）
- AI 替代人工的成本节省（年省 ¥23-35 万/塘）
